# Typescript(expressJs)

> npm i --save-dev typescript
> npx tsc --init
> npm i express
> npm i --save-dev @types/express
> npx tsc - to create .js file
> npm i --save-dev nodemon
> npm install -D ts-node
> npx nodemon src/server.ts - to run environment process
> npm i -D eslint
> npx eslint --init
